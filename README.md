Setup instructions
------------------

First you need to [install yarn](https://yarnpkg.com/en/docs/install)

```
yarn
```

### For development:

* *Step 1.* Copy `config/client.json.example` to `config/client.json`
* *Step 2.* Change config to your needs.
* *Step 3.* Run command `yarn dev`
